20k-40k

职位描述

职位诱惑：

鹅厂福利，有竞争力的薪资，广阔的发展空间

 

职位描述：

主要负责腾讯云国际业务的本地化相关工作。

1.负责本地化结算的前端架构与业务开发；

2.负责内部工具、运营系统建设，追求更高研发效率；

3.负责对新技术

 

岗位要求：

1. 本科及以上学历，计算机相关专业，5 年以上 Web 前端开发经验；

2. 熟悉 HTTP 基本原理，能熟练运用 HTML、CSS、JavaScript 构建高性能的 Web 应用程序；

3. 熟悉原生 JavaScript，熟悉 jQuery，React、Vue 等框架及类库；

4. 有使用 grunt、gulp、webpack 等工具进行前端工程化经验；

5. 工作积极主动，具有良好的逻辑思维及语言表达能沟通力和团队协作精神。

工作地址：
深圳 - 南山区 - 卓越城梅林中心广场北区一期2栋1301C

公司简介

一家以家电制造业为主的大型综合性企业集团，于2013年9月18日在深交所上市，旗下拥有小天鹅（SZ000418）、威灵控股（HK00382）两家子上市公司。IT为美的集团总部一级职能部门，是美的集团全球业务流程信息化实现主体，专注IT能力和系统建设。现有技术人员近1000人，团队年轻化且实力雄厚，既有来自全球的IT精英，又有来自世界顶... 根据业务模式发展需要，推动全面数字化的“632”全流程IT系统建设，并辅以五大IT举措（智能制造、大数据、移动化、美的云、全球化），为企业提供高效服务。 未来IT将推行数字化2.0，在智能制造、全球定制、区块链、云计算以及人工智能等领域进行研究和突破，不断将成熟的新技术应用于企业战略运营中。集团信息化建设，推动632项目流程变革与IT解决方案落地，培育IT系统自主研发和自主交付能力。
